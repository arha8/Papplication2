Практический проект. Вариант приложения с использованием MVC
(добавил controller, заменил JerseyServlet на DispatcherServlet, добавил Mapper и ViewResolver(xml-файл), поменял конвертер LocalDate (c XMLAdapter на StdConverter).

Описание:
Имеет аналогичный функционал с приложением Papplication, но реализован вариант с использованием Spring MVC.
Представляет из себя Spring MVC-приложение, с подключённой базой данных PostgreSQL. Отображение в браузере выполнено через JSP (с использованием JSTL). Также реализован пример использования Web-сервлета.
Через JSON можно обратиться к базе и получить ответ так же в формате JSON, к url-адресам привязаны JSP-страницы с отображением ответов на некоторые запросы к базам данных.

В работе использовал: Spring MVC, Sring Data Jpa, Spring Web, JUnit, Java EE (вариант с отображением через сервлет), JSP+JSTL.
Настройка спринга через xml-файл.
В файле order.sql находятся SQL-команды создания таблиц  и внесения в них значений.
